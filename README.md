# SQL Online Book Store Project

## Project Overview
This project is a **SQL-based Online Book Store database system** designed to practice real-world SQL queries using a relational database.

The main goal of this project is to strengthen **SQL fundamentals** such as joins, filtering, aggregation, and subqueries.

---

## Database Structure
The database consists of the following tables:

### 1. Books
- Book ID
- Title
- Author
- Genre
- Price
- Stock Quantity

### 2. Customers
- Customer ID
- Name
- Email
- City
- Country

### 3. Orders
- Order ID
- Customer ID
- Book ID
- Order Date
- Quantity
- Total Amount
